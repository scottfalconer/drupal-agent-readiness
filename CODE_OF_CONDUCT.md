# Code Of Conduct

This project expects direct, evidence-based, respectful collaboration.

- Challenge claims, rubrics, prompts, evaluators, and interpretations with
  concrete evidence.
- Keep disagreements focused on artifacts and reproducibility.
- Do not post private site data, secrets, credentials, or personal information.
- Do not harass, threaten, or target contributors.

Report conduct concerns through a GitHub issue if the report can be public. For
private reports, contact the maintainer listed in `SECURITY.md`.

Maintainers may remove comments, issues, or pull requests that violate these
expectations.
